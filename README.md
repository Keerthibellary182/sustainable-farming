# sustainable-farming
Problem Statement:
Plant Disease Detection using Convolutional Neural Networks (CNN)

Agriculture is a major contributor to the global economy, and healthy crop production is essential for food security. However, plants are often affected by various diseases that lead to reduced yield and economic loss. Traditional methods of disease detection rely on manual observation by farmers or experts, which is time-consuming, subjective, and often inaccurate.
To address this challenge, this project aims to develop an automated plant disease detection system using Convolutional Neural Networks (CNN). The model will classify plant leaves as healthy or diseased, and if diseased, it will identify the type of disease based on visual symptoms in the leaf image.
By leveraging deep learning, the system can assist farmers and agricultural experts in early diagnosis, timely treatment, and sustainable crop management, ultimately promoting eco-friendly and resource-efficient farming.
